# Разработка интернет-приложений (РИП) 2025

## Тема: Расчет TTR-авторов книг

Объектом разработки является система анализа лексического разнообразия текстов на основе коэффициента TTR (Type-Token Ratio). Данный программный модуль позволяет лексическое разнообразие автора как отношение количества уникальных слов (типов) к общему числу слов (токенов) в предоставленных текстах.
